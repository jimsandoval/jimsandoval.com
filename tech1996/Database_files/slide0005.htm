lACID Criteria
–Atomicity - Multiple inserts and updates are tied into a single transaction
–Consistency - Transaction is complete; it is never in a half-way state
–Isolation - Transaction doesn’t affect other un-related things
–Durability - Transaction can’t be lost in the system
